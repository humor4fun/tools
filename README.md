# tools

A collection of small, standalone browser-based tools by Chris Holt. Each tool lives in its own subdirectory and is entirely self-contained — no dependencies to install, no build step, no server required.

---

## Tools

| Tool | Description |
|------|-------------|
| [text](./text/) | Single-file local file browser and editor. Open any folder of text files and browse, read, and edit them directly in the browser using the File System Access API. |
| [list](./list/) | List sort, clean, and transform tool. Sort by name, date, or length; deduplicate; relabel; prune; find and replace; change case — all in a single textarea. |
| [transcode](./transcode/) | Text encoding, decoding, and hashing tool with a chainable pipeline UI. Compose multi-step transformations: Base64, URL encoding, HTML entities, hex, binary, ROT13, Morse, SHA, MD5, and more. |
| [barcode](./barcode/) | Generate 1D barcodes in Code 128, EAN-13, UPC-A, Code 39, and ITF formats. Customize colors, size, label, and rounded corners. Export as PNG or SVG. |
| [qr](./qr/) | Generate QR codes and Data Matrix codes. Choose pixel style, overall shape, error correction level, and optionally embed a center logo. Export as PNG or SVG. |
| [join](./join/) | Merge PDFs, images, and text files into a single PDF. Drag to reorder, set delimiters between content, and customize output. All processing happens client-side. |
