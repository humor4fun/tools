# tools

A collection of small, standalone browser-based tools by Chris Holt. Each tool lives in its own subdirectory and is entirely self-contained — no dependencies to install, no build step, no server required.

---

## Tools

| Tool | Description |
|------|-------------|
| [mdweb](./mdweb/) | Single-file Markdown folder reader and editor. Open any local folder of `.md` files and browse, read, and edit them directly in the browser. |
