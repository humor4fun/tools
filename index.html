tools.bluejar.org

Small tools.
No install required.

A collection of standalone browser-based utilities by Chris Holt. Each tool is self-contained — no dependencies, no build step, no server.

Tools

text

Open any local folder of text files and browse, read, and edit them in the browser. Renders Markdown, highlights code, and previews HTML — all offline after first load.

markdown · editor
transcode

Encode, decode, hash, and transform text. Base64, URL encoding, HTML entities, hex, binary, ROT13, Morse code, SHA hashing, and CIRCL file hash lookup — all offline.

encode · decode · hash
list

Sort, clean, transform, and label any list. Supports alphabetical, numeric, date, name, and random ordering — plus case conversion, deduplication, find/replace, and more.

sort · list · text
barcode

Generate 1D barcodes in Code 128, EAN-13, UPC-A, Code 39, and ITF. Customize colors, size, label, and rounded corners. Export as PNG or SVG.

barcode · generator
qr

Generate QR codes and Data Matrix codes. Choose pixel style, overall shape, error correction level, and optionally embed a center logo. Export as PNG or SVG.

qr · datamatrix · 2d
